#Prog2 Project - Soyuz

This project started with a #simple# idea, lets make a phone from an old radio. 

##Hardware

* LILYGO TTGO T-CALL ESP32 SIM800L
* 1602 LCD Display
* I2C LCD Driver
* BME280 Temperature and humidity sensor
* Rotary encoder
* Push button
* 2 x Rocker switch
* 4 x White led
* 4 x 470 ohm resistor
* 2 x 220 kohm resistor
* Microphone
* 8 ohm speaker
* Li-ion battery charger
* 3 x 18650 battery
* 3 x 18650 battery case

##Needed for assembly

* Mounting material
	* Wire
	* Solder wire
	* Insulating tape
	* Heatshrink
* Tools
	* Soldering iron
	* Wire cutter
